CSS Basics

CSS (Cascading Style Sheets) is a language used for describing the look and formatting of a document written in HTML. Here's a brief overview:

  • Selectors: CSS selectors define the elements to which a set of CSS rules apply.
  • Box Model: All HTML elements can be considered as boxes, consisting of content, padding, border, and margin.
  • Positioning: The position property specifies the type of positioning method used for an element (static, relative, fixed, absolute, or sticky).

For more details, visit the CSS documentation on W3C or check MDN Docs.
